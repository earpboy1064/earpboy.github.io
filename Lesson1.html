Hello World

Lesson 1


The first thing we are going to learn is how to get the computer to print out whatever message we want. In order to do this we must open our chosen programming environment and create a new python file. This file will be where we will write our first program. To start off we need to define what we want the program to do. This can be done on paper or in your head if you already have a good idea of what you want to accomplish. Our goal here is to print the words “Hello world” to the console. This printing will be done through the use of strings. The best way to understand strings is by looking at the words on this page. A string is a combination of usually two or more characters. Strings can be used to contain single words or an entire sentence or even an entire book. In our program our string will contain the words “Hello world” so the computer will know what to print. To get started lets define our string,
output = "Hello world"
The name of our string is output, this is how we will tell the computer what it should print.
print(output)
After we have made our new string and defined what it contains, we will then tell the computer to print our string. If we check our Terminal we should be able to see our string has been printed.
Hello world
