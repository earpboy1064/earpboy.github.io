Interest Calculator

Lesson 3


Variables are a very important concept to understand, and that is why we will write a small program using several different types of variables to further help us learn how they work and to improve our comfort with them.

We will write a simple interest calculator that will print the amount of interest generated over a period of time. I encourage you to go off and try this yourself first and then come back to this lesson to see how close you were able to get.

first thing we will do is create variables to hold the interest amount. We will then need the user to input the amount of money and the total time that has passed in order for us to calculate the interest.
#include<iostream>
#include"string.h"

namespace std;

int main()
 {
float interest = 0.0;
 float moneyOwed = 0.0;
 int timePassed = 0;
 float totalInterestAccrued = 0;
We will now collect user input.
cout << "Please Type the interest percentage: "<< endl;
 cin >> interest;
 cout << "Please type the amount of money owed: "<< endl;
 cin >> moneyOwed;
 cout << "Please type the amount of time passed in months: " << endl;
 cin >> timePassed;
Now that we have all of the data we need we write the formula to calculate the interest.
interest = interest / 100;
totalInterestAccrued = (interest * moneyOwed) * timePassed; }
}

Once we have the amount of interest accrued we can print the it so the user can see the amount of interest owed.
cout << "The total Interest Accrued is $" << totalInterestAccrued;

We can now run our program and input the values for each variable as if we were the intended user.

With the completion of this program you have written your first real program!
