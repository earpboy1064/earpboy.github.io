Interest Calculator

Lesson 3


Variables are a very important concept to understand, and that is why will write a small program using several different types of variables to further help us learn how they work and to improve our comfort with them.

We will write a simple interest calculator that will print the amount of interest generated over a period of time. I encourage you to go off and try this yourself first and then come back to this lesson to see how close you were able to get.

first thing we will do is define the variables. We will then need the user to input the amount of money, interest rate, and the total time that has passed in order for us to calculate the interest. in order to collect the user input in Java we must import the java utility Scanner. We will then need to create a new scanner object which we can use to read the users input from the terminal.

Because we are using the java Scanner we must remember that the scanner will always return a string. But for this exercise we will need floats and integers so we can do the arithmetic needed to calculate the interest. In order for us to convert the strings to the appropriate variable we can type Float.parseFloat(in.nextLine()) or Integer.parseInt(in.nextLine()) to collect and convert the user input to the appropriate type.
import java.util.Scanner;

class javaLesson3 {

{
public static void main()
 {
float interest;
 float moneyOwed;
 int timePassed;
 float totalInterestAccrued;
 System.out.println("Please Type the interest percentage: ");
 interest = Float.parseFloat(in.nextLine());
 System.out.println("Please type the amount of money owed: ");
 moneyOwed = Float.parseFloat(in.nextLine());
 System.out.println("Please type the amount of time passed in months: ");
 timePassed= Integer.parseInt(in.nextLine());
Now that we have all of the data we need we write the formula to calculate the interest.
interest = interest / 100;
totalInterestAccrued = (interest * moneyOwed) * timePassed;
Once we have the amount of interest accrued we can print the it so the user can see the amount of interest owed.
System.out.println("The total Interest Accrued is $" + totalInterestAccrued);

We can now run our program and input the values for each variable as if we were the intended user.
}
}


With the completion of this program you have written your first real program!
