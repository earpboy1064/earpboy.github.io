Conditional statements

Lesson 4


The next important topic to learn is conditional statements. conditional statements allow the program to make a simple decision based on a defined reason. There exists several different ways to write conditional statement but for this example we will focus on the statements "if" and "else". We will use if and else to allow our interest calculator to tell the user if they have inputted incorrect values. The if statement will check to see if the value of the total accrued interest is greater then 0 and if it is not it will print an error message. The "else" will be used to signify what code should be run if the total accrued interest is actually greater than 0. To begin we need to load up our previously written interest calculator.
#include<iostream>
#include"string.h"

namespace std;

int main()
 {
float interest = 0.0;
 float moneyOwed = 0.0;
 int timePassed = 0;
 float totalInterestAccrued = 0;
We will now collect user input.
cout << "Please Type the interest percentage: "<< endl;
 cin >> interest;
 cout << "Please type the amount of money owed: "<< endl;
 cin >> moneyOwed;
 cout << "Please type the amount of time passed in months: " << endl;
 cin >> timePassed;
Now that we have all of the data we need we write the formula to calculate the interest.
interest = interest / 100;
totalInterestAccrued = (interest * moneyOwed) * timePassed; }
}


Once we have the amount of interest accrued we can use the if statement to determine if the values entered by the user are valid
We can see below that the if statement is checking if the total interest accrued is either less then or equal to 0.
If it is then the following code will be run
if (totalInterestAccrued <= 0)
{
cout << "ERROR" << endl;
cout << "Please try running the program again with the correct values" << endl;
}

If the value for total interest accrued is not less than or equal to 0 then the code below the else will execute

else
{
cout << "The total Interest Accrued is $" << totalInterestAccrued << endl;
}
cout << "The total Interest Accrued is $" << totalInterestAccrued << endl;

}
}

We can now run our program and input the values for each variable. To check if the if statement is working you can type a 0 for any of the variable and you will see the printed error message.
