introduction to Variables

Lesson 2


Now that we are familiar with strings we will learn about the concept of variables. A variable is a container that can contain different types of data. A variable type that you are now familiar with is the string, the string only contains characters. Other types of variables include integer which is any whole number, and a float, which is a whole number that also has decimal places. In this lesson we will practice creating and using these variables. First we will create two new integer variables.
int number_one = 10
int number_two = 7
Now that we have two variables, we will then practice using them. First, we will add and store the result in a new variable.
int number_three = number_one + number_two
We will now print this new variable.
cout << number_three
And look at the output in our terminal
17
